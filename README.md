# Ayush Singh Pawar — Portfolio

Single-page modern portfolio built with plain HTML/CSS/JS.

## 📁 Project Directory Structure
Here is the layout and directory structure of this repository:
```text
MyPortfolio/
├── .git/                      # Git version control directory
├── Ayush.jpg                  # Profile hero showcase image
├── Full Stack Certificate.jpg # Software developer internship certificate
├── README.md                  # Comprehensive project documentation
├── index.html                 # Main website markup & layout structure
├── resume.pdf                 # Professional resume download asset
├── script.js                  # Intersection observers, dynamic year & contact form handling
└── styles.css                 # Slate & Emerald custom CSS theme tokens & design system
```

## ✨ Key Features
* **Slate & Emerald Theme:** Clean, state-of-the-art dark mode using custom HSL/RGB variables with a glowing neon accent layer.
* **Modern Typography:** Sophisticated pairing of **Space Grotesk** (geometric, bold headers) and **Inter** (highly readable sans-serif body text).
* **Responsive Widescreen Hero:** Elegant desktop layout that refines dynamically into side-by-side card sections on widescreen mobile landscape.
* **Scroll-Reveal System:** Light weight Intersection Observer API driving smooth, native fading animations as sections come into view.
* **Smart Contact Handling:** Modern dynamic email client loader paired with an instant copy-to-clipboard backup system.

## Run
- Open `index.html` directly in your browser, or
- In VS Code, use a Live Server extension (recommended) and open the generated local URL.

## Customize
Edit these files:
- `index.html` — content/sections
- `styles.css` — theme + layout + animations
- `script.js` — scroll-reveal + footer year

## Add your photo
- Put your profile image in this folder and name it `Ayush.jpg`
- Recommended: square image (e.g. 800×800) for best crop
